# 🔒 セキュリティ通知

## 依存関係の脆弱性について

### 現状
- **hull.js** に重大な脆弱性（コードインジェクション）が存在
- scratch-vm → scratch-render → hull.js の依存関係チェーンで発生
- 公式のscratch-vmパッケージに含まれているため、直接修正不可

### リスク評価
- **影響度**: 低〜中
- **理由**: 
  - hull.jsは描画ライブラリの内部で使用
  - 外部からの直接的な攻撃経路は限定的
  - ローカル環境での教育用途が主な使用場面

### 対策
1. **短期**: 現状のまま使用（リスク受容）
   - 教育環境での限定的な使用
   - 信頼できるユーザーのみアクセス

2. **中期**: 代替実装の検討
   - scratch-vmを使わない独自実装
   - または脆弱性が修正されたフォーク版の使用

3. **長期**: 公式修正待ち
   - scratch-vm の更新を定期的にチェック
   - 修正版がリリースされ次第更新

### 推奨事項
- プロダクション環境では使用しない
- ファイアウォールで保護された環境で使用
- 定期的な依存関係の更新チェック
