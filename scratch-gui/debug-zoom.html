ズームコントロール デバッグ

ブラウザのコンソールを開いて、以下のコードを実行してください：

// メインワークスペースのズームコントロールのパスを確認
const mainZoomImages = document.querySelectorAll('.blocklyZoom image');
console.log('=== メインズームコントロール ===');
mainZoomImages.forEach((img, i) => {
    const href = img.getAttributeNS('http://www.w3.org/1999/xlink', 'href');
    console.log(`Image ${i}: ${href}`);
});

// ワークスペースのpathToMediaを確認
if (window.Blockly && window.Blockly.mainWorkspace) {
    console.log('=== Workspace pathToMedia ===');
    console.log('Main workspace:', window.Blockly.mainWorkspace.options.pathToMedia);
    
    const flyout = window.Blockly.mainWorkspace.getFlyout();
    if (flyout) {
        const flyoutWs = flyout.getWorkspace();
        console.log('Flyout workspace:', flyoutWs.options.pathToMedia);
    }
}

// 実際のズームコントロール要素を確認
console.log('=== 全てのズームコントロール ===');
document.querySelectorAll('.blocklyZoom').forEach((zoom, i) => {
    console.log(`Zoom control ${i}:`, zoom);
    const images = zoom.querySelectorAll('image');
    images.forEach((img, j) => {
        console.log(`  Image ${j}:`, img.getAttributeNS('http://www.w3.org/1999/xlink', 'href'));
    });
});

修正方法

コンソールで確認した正しいパスを使って修正します。
