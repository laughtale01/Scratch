## 🚀 テスト実行ガイド

### 自動テストの実行

```bash
# JavaScriptテスト全実行
cd tests
node run-all-tests.js

# Javaテスト全実行
cd minecraft-mod
./gradlew test

# 個別テスト実行例
node tests/load-test.js
./gradlew test --tests "*.BuildingCommandsTest"
```

### 手動テストの実行

1. **環境準備**
   - Minecraft 1.20.1 + Modの起動
   - Scratch拡張の読み込み

2. **基本動作確認**
   - 接続テスト
   - 各ブロックの動作確認
   - エラーケースの確認

3. **シナリオテスト**
   - 授業想定フローの実行
   - 複数人での協調作業
