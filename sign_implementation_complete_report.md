## 機能詳細

### レポーターブロックへの統合

看板は既存の「装飾ブロック」レポーターブロックに自動的に追加されます：

```scratch
// Scratchブロック
blockDecoration
  ↓
  [オークの看板 ▼]  ← 11個の看板が選択可能
```

**使い方:**
1. 「装飾ブロック」レポーターブロックをクリック
2. ドロップダウンから看板を選択
3. 「ブロックを置く」コマンドで配置

### Minecraft互換性

**対応バージョン:**
- Minecraft 1.14+ (木材別看板の導入)
- Minecraft 1.16+ (ネザー木材看板)
- Minecraft 1.19+ (マングローブ、竹看板)
- Minecraft 1.20+ (サクラ看板)

**看板の特徴:**
- テキストを4行まで書き込み可能
- 染料で文字色を変更可能
- 輝くイカスミで文字を発光可能
- 地面、壁面に設置可能

## 技術的詳細

### 実装方法

#### 1. DECORATION_BLOCKSへの追加
```javascript
const DECORATION_BLOCKS = [
  // ... 既存の装飾ブロック81個 ...
  {text: 'オークの看板', value: 'oak_sign'},
  {text: 'トウヒの看板', value: 'spruce_sign'},
  // ... 11個の看板 ...
];
```

#### 2. sorted_categories.jsonへの追加
```json
{
  "DECORATION_BLOCKS": [
    // ... 既存ブロック ...
    {"text": "オークの看板", "value": "oak_sign"},
    {"text": "トウヒの看板", "value": "spruce_sign"}
    // ... 11個の看板 ...
  ]
}
```

#### 3. 自動レポーター統合
既存の`blockDecoration`レポーターブロックが自動的に新しい看板を含むため、
追加のコード変更は不要。
