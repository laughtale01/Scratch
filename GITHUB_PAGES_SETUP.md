## ❓ トラブルシューティング

### 問題1: 「接続エラー」と表示される
**原因**: Minecraftが起動していない、またはModが読み込まれていない
**解決**: 
1. Minecraftを再起動
2. コンソールで「WebSocket server started on port: 14711」メッセージ確認
3. ページを再読み込み

### 問題2: 拡張機能ボタンが反応しない  
**原因**: JavaScript無効またはブラウザ互換性
**解決**: 
1. ブラウザのJavaScriptを有効化
2. 別のブラウザで試行
3. ページを完全に再読み込み（Ctrl+F5）

### 問題3: ブロックが置けない
**原因**: 座標エラーまたは権限不足
**解決**: 
1. プレイヤーの現在位置確認（F3キー）
2. ゲームモードをクリエイティブに変更
3. 座標値を調整
