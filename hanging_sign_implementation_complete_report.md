## 機能詳細

### レポーターブロックへの統合

吊り看板は既存の「装飾ブロック」レポーターブロックに自動的に追加されます：

```scratch
// Scratchブロック
blockDecoration
  ↓
  [オークの看板 ▼]
  [トウヒの看板 ▼]
  ...
  [歪んだ看板 ▼]
  [オークの吊り看板 ▼]  ← NEW! 11個の吊り看板が追加
  [トウヒの吊り看板 ▼]  ← NEW!
  ...
  [歪んだ吊り看板 ▼]    ← NEW!
```

**使い方:**
1. 「装飾ブロック」レポーターブロックをクリック
2. ドロップダウンから吊り看板を選択
3. 「ブロックを置く」コマンドで配置

---

### 吊り看板の特徴（通常の看板との違い）

#### 設置場所の拡張
**通常の看板:**
- 地面（床）
- 壁面

**吊り看板:**
- ブロックの側面
- ブロックの下面（吊り下げ）
- フェンスの下
- 壁の下
- 鎖の下
- 他の吊り看板の下

#### 装飾性の向上
- より多様な設置方法
- お店の看板演出に最適
- 天井から吊り下げる案内板
- 連続設置で長い文章表示

#### 共通機能
- テキスト4行記入可能
- 染料で文字色変更
- 輝くイカスミで文字発光
- ミツロウで編集ロック

## 技術的詳細

### 実装方法

#### 1. 論理的グループ化
通常の看板の直後に吊り看板を配置することで、以下の利点:
- ドロップダウンメニューで看板が連続表示
- ユーザーが見つけやすい
- 木材の種類が対応（oak_sign → oak_hanging_sign）

#### 2. コード実装
```javascript
const DECORATION_BLOCKS = [
  // ... 既存の装飾ブロック80個 ...

  // 通常の看板（11個）
  {text: 'オークの看板', value: 'oak_sign'},
  {text: 'トウヒの看板', value: 'spruce_sign'},
  // ... 11個 ...

  // 吊り看板（11個）
  {text: 'オークの吊り看板', value: 'oak_hanging_sign'},
  {text: 'トウヒの吊り看板', value: 'spruce_hanging_sign'},
  // ... 11個 ...
];
```

#### 3. 自動レポーター統合
既存の`blockDecoration`レポーターブロックが自動的に新しい吊り看板を含むため、
追加のコード変更は不要。
