# 🎮 はじめての人のための Minecraft×Scratch 使い方ガイド

## 📖 このシステムについて

**Minecraft協調学習システム**は、人気ゲーム「Minecraft」とプログラミング学習ツール「Scratch」を組み合わせた、小学生向けの学習システムです。

### 🌟 できること
- Scratchのブロックを使ってMinecraftを操作
- 友達と一緒に建築やプログラミング
- 楽しみながらプログラミングの基礎を学習

---

## 🚀 かんたんスタートガイド

### 準備するもの
1. **パソコン**（Windows 10以上）
2. **マインクラフト Java版**（バージョン1.20.1）
3. **インターネット接続**（ダウンロード用）

## 📥 インストール手順（保護者・先生向け）

### ステップ1: 必要なソフトをインストール

#### 1-1. Java 17をインストール
1. [こちら](https://adoptium.net/)からダウンロード
2. 「Windows x64」を選んでダウンロード
3. ダウンロードしたファイルをダブルクリックしてインストール

#### 1-2. Node.jsをインストール
1. [こちら](https://nodejs.org/)からダウンロード
2. 「LTS版」を選んでダウンロード
3. ダウンロードしたファイルをダブルクリックしてインストール

### ステップ2: システムファイルをダウンロード
1. 配布されたZIPファイルをダウンロード
2. デスクトップに展開（解凍）
3. 「minecraft_collaboration_project」フォルダができます

### ステップ3: 初回セットアップ
1. 「minecraft_collaboration_project」フォルダを開く
2. 「scripts」フォルダの中の「windows-setup.bat」をダブルクリック
3. 黒い画面が出て、自動的にセットアップが始まります
4. 「Setup completed!」と表示されたら完了

## 🎮 使い方（子ども向け）

### 1. Minecraftを起動する

#### かんたんな方法
1. 「scripts」フォルダを開く
2. 「run-minecraft.bat」をダブルクリック
3. Minecraftが自動で起動します

#### いつもの方法
1. Minecraft Launcherを起動
2. 「Forge 1.20.1」を選ぶ
3. 「プレイ」をクリック

### 2. Scratchを開く

#### 方法1: かんたんページ
1. プロジェクトフォルダの「public」を開く
2. 「index.html」をダブルクリック
3. ブラウザでScratchが開きます

#### 方法2: フル機能版
1. 「scripts」フォルダを開く
2. 「start-extension-server.bat」をダブルクリック
3. ブラウザで http://localhost:8601 を開く

### 3. つなげる

1. Scratchで「拡張機能」ボタンをクリック（左下の青いボタン）
2. 「Minecraft Controller」を選ぶ
3. 「🔌 Minecraftに接続する」ブロックをドラッグ
4. 緑の旗をクリックして実行

### 4. 遊んでみよう！

#### 基本のブロック

**ブロックを置く**
```
🧱 [stone]を X:[0] Y:[64] Z:[0] に置く
```

**チャットする**
```
💬 チャット: [こんにちは！]
```

**自分の位置を知る**
```
📍 プレイヤーのX座標
📍 プレイヤーのY座標
📍 プレイヤーのZ座標
```

## 🎨 楽しいプログラム例

### 例1: 石の塔を作る
```scratch
🔌 Minecraftに接続する
くりかえし (10) 回
  🧱 [stone]を X:[0] Y:[64 + カウンター] Z:[0] に置く
  1秒待つ
```

### 例2: カラフルな床
```scratch
🔌 Minecraftに接続する
くりかえし (5) 回
  くりかえし (5) 回
    🧱 [wool]を X:[カウンター1] Y:[64] Z:[カウンター2] に置く
```

### 例3: 友達を招待
```scratch
🔌 Minecraftに接続する
📧 [友達の名前]さんを招待
💬 チャット: [一緒に遊ぼう！]
```

## ❓ 困ったときは

### よくある質問

**Q: 「接続できません」と出る**
A: 
1. Minecraftが起動しているか確認
2. 「scripts」→「run-minecraft.bat」で再起動
3. それでもダメなら先生に相談

**Q: ブロックが置けない**
A: 
1. ゲームモードが「クリエイティブ」か確認
2. 座標（X,Y,Z）が正しいか確認
3. 接続されているか確認

**Q: 友達と遊べない**
A: 
1. 同じWi-Fiにつながっているか確認
2. お互いにMinecraftを起動しているか確認
3. 招待を送った後、相手が承認するまで待つ

### エラーメッセージ対応表

| メッセージ | 意味 | 対処法 |
|-----------|------|--------|
| ❌ Minecraftに接続できません | Minecraftが起動していない | Minecraftを起動する |
| ⏰ 接続がタイムアウトしました | 接続に時間がかかりすぎ | もう一度試す |
| ❓ その友達は見つかりませんでした | 友達の名前が間違っている | 正しい名前を入力 |
| 🚫 このブロックは使えません | 危険なブロック（TNTなど） | 別のブロックを使う |

### 安全機能
- ローカルネットワーク内のみで動作
- 危険なブロック（TNT、溶岩など）は自動的にブロック
- 緊急帰宅機能で安全に自分のワールドに戻れます

- **創造性**: 3D空間での表現力を育成
- **問題解決**: エラーを自分で解決する力
